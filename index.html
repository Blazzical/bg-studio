Whole-image filters

Add

Vector shapes

Pen tool (P): click to drop anchor points, click-drag to pull out a curve. Click the first point to close. Esc/Enter finishes · Backspace removes the last point.

Brush

Brush applies to the selected image / paint layer. Erase = manually rub out the background. [ / ] resize · B toggles.

Selected layer

Click a layer on the canvas to select it.
Swatches
Click a swatch to apply to the last-touched colour · shift-click removes it. “Save” stores this text’s full style; “Default” = new text starts from it.
Click a cell to select · Shift-click another for a range.
Cell bg
Cell text
Drag the shape to move · corner handles scale/rotate. Line & Path: drag anchor points; Alt-drag an anchor to pull out curve handles (Alt-click a smooth anchor to make it a corner); drag a round handle to reshape (Alt = break it independently). Double-click a Path segment to add a point, a point to remove it.
Filters (this layer)
Non-destructive except AI portrait blur, which bakes into the layer (use Restore layer to revert).

Layers

    Drag to move · corner handles to scale · top handle to rotate · drag a canvas edge to crop · paste (Ctrl/Cmd+V) an image as a layer · Del to delete
